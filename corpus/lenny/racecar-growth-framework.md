# The Racecar Growth Framework — Lenny Rachitsky
**Source:** https://www.lennysnewsletter.com/p/the-racecar-growth-frameworkexpanded
**Co-developed with:** Dan Hockenmaier (Faire, Thumbtack, Reforge)
**Type:** Comprehensive growth strategy framework

## Overview
All growth strategy summarized in one framework with four components:

## 1. Growth Engine (Most Important — Self-Sustaining Loop)
The only component that can be self-sustaining — creating output that reinvests into more growth.

### Four Growth Engines:
1. **SEO:** Users create content → People discover via search → Become users → More content
2. **Paid Ads:** Run ads → Drive users → Generate revenue → Fund more ads
3. **Sales:** Hire salespeople → Get customers → Drive revenue → Hire more salespeople
4. **Virality:** User signs up → Enjoys & shares product → Friends sign up → They share too

## 2. Kickstarts (Unscalable Tactics for First 1,000 Users)
Nine most common kickstarts:
1. Share with friends and colleagues
2. Reach out to targeted strangers (cold DM, email, phone, door-to-door)
3. Go where target audience hangs out (online communities, offline locations)
4. Enlist influencers (paid or organic)
5. Get press
6. Create viral content (blog, video, one-off website)
7. Get physical placement (flyers, stickers, signs)
8. Host an event
9. Be first on a platform

## 3. Turbo Boosts (One-Off Growth Accelerators)
Temporary events that accelerate growth but don't sustain. Overlap with kickstarts but can work at any scale.

## 4. Lubricants (Efficiency Improvements)
Make existing growth engine run more efficiently.

## When to Focus on Each:
- **Just starting:** Kickstarts + some Turbo boosts until Growth engine drives majority
- **Working engine:** Invest in Lubricants + occasional Turbo boosts
- **Meaningful scale:** Explore Mid-stage accelerants
- **Before asymptote:** Experiment with additional Growth engine

## B2C vs B2B Differences
- B2C: Multiple kickstart options available
- B2B: Really only three — friends/colleagues, pitch where they hang, press
- B2B growth engine: Almost always sales (some with product-led element)
